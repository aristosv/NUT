# NUT
Network UPS Tools

## install
clone repo & run install
```
git clone https://github.com/aristosv/network-ups-tools.git
bash ~/network-ups-tools/operations/install
```

## access
```
http://computer_ip/cgi-bin/nut/upsstats.cgi
```

## update
run update script
```
~/network-ups-tools/operations/update
```
